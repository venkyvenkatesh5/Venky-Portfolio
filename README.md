# Portfolio Website

A personal portfolio website built using **HTML, CSS, and JavaScript** to showcase projects, skills, and experience.

## Features
- **Responsive Design** – Works on all devices (mobile, tablet, and desktop)
- **Smooth Animations** – Enhances user experience
- **Interactive UI** – JavaScript for dynamic content
- **Project Showcase** – Section to display past work
- **Contact Form** – Users can reach out via email

## Technologies Used
- **HTML** – Structure of the website
- **CSS** – Styling and layout (includes Flexbox & Grid)
- **JavaScript** – Adds interactivity

## Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/venkyvenkatesh5/Venky-Portfolio
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in a browser.

## Usage
- Modify `index.html` to add your content.
- Customize styles in `style.css`.
- Add dynamic functionality in `script.js`.

## Folder Structure
```
portfolio/

│-- assets/
```
